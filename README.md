# CRTI · 伴侣关系类型测试

**Couple Relationship Type Indicator**

一个单页测试，帮你和伴侣看清你们的关系模式。16种类型，32道题，纯前端，无后端，无需登录。

---

## 怎么用

单人作答，以"我们"的视角回答题目。两个人可以各自测一份，再对照结果——看你们对这段关系的认知是否一致，往往比结果本身更有意思。

---

## 类型体系

16种类型由两个轴交叉组成：

**轴一：关系基础类型**（你们怎么互动）
- ALLY 联盟型 · PULL 拉扯型 · ROLE 分工型 · LEAN 失衡型

**轴二：主导结构**（谁在定义关系的运行方式）
- HIM 男方主导 · HER 女方主导 · DUAL 双主导 · HIDE 隐性主导

结果页还包含三个指数条：决策权、情绪主导、现实承担。
